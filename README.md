# About

## This website is a professional portfolio designed to showcase my journey as a Fullstack Developer and Systems Development Technician. It focuses on high-performance UI/UX, real-time interactivity, and technical depth.
Technologies
### Core

    HTML5 & CSS3: Modern structure utilizing CSS Variables for theme management and Flexbox/Grid for layout.

    JavaScript (ES6+): Powers the core logic, dynamic modal systems, and DOM manipulation.

    Node.js: Underlying technology for several listed projects and backend integration.

### Visuals & Graphics

    Three.js: A cross-browser JavaScript library and API used to create and display animated 3D computer graphics in a web browser using WebGL.

    Vanta.js (Net variant): An animated background library built on top of Three.js that provides the interactive polygon/constellation effect.

    GSAP (GreenSock Animation Platform): Used for high-performance, professional-grade web animations.

    AOS (Animate On Scroll): Provides the scroll-triggered animations that reveal sections as you move down the page.

### Design & Assets

    Font Awesome: Comprehensive icon library used for the navigation bar, project tags, and contact links.

    Google Fonts (Inter): High-legibility typography optimized for modern user interfaces.

    Glassmorphism UI: Design style utilizing backdrop-filter for translucent, frosted-glass effects on cards and navigation.

### CDNs & Package Managers

    UNPKG: Fast, global content delivery network for everything on npm, used for AOS.

    Cloudflare CDNJS: Hosting for essential libraries like Three.js and Font Awesome.

    JSDelivr: A free, fast, and reliable Open Source CDN for npm and GitHub, used for Vanta.js.
